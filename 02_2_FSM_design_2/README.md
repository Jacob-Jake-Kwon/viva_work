# FSM Design 2

Additional FSM examples covering simple state transitions, output generation, and Mealy-style sequence detection.

## Modules

| Module | Description |
| --- | --- |
| `FSM_ex1` | Minimal two-state FSM example. |
| `FSM_ex2` | Three-state FSM with registered state and combinational output. |
| `FSM_ex3` | Sequence-oriented FSM example. |
| `seq_det_mealy` | Mealy sequence detector. |

## Files

All source files are under `rtl/`.

## Notes

These modules are intended for HDL practice and simulation. Add a board wrapper and constraints only if mapping one of them to FPGA pins.
