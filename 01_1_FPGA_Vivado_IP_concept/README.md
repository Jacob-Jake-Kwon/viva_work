# FPGA Vivado IP Concept

This folder documents the Vivado IP workflow used throughout the later FPGA examples. The matching PDF is concept-focused and does not provide standalone RTL or XDC source files.

## What This Covers

- Vivado IP Integrator basics
- IP block reuse and module instantiation concepts
- How generated wrappers become the top-level hardware interface
- How later Basys3 examples connect RTL/IP blocks to board pins through XDC constraints

## Source Files

No HDL or XDC files are required for this concept-only section.

## Notes

Use this folder as the overview/reference section before opening the hands-on folders such as `01_2_Vivado_IP_LED_blink` and `01_3_Stopwatch_RTL_IP`.
